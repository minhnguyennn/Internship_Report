Lệnh ancestors
		Bạn có thể sử dụng MyClass.ancestors để xem danh sách các class và module trong chuỗi thừa kế của MyClass.
		
		class A
		end

		module B
		end

		class C < A
		  include B
		end

		puts C.ancestors
Cách sử dụng modules
		Một mô-đun cho phép chúng tôi nhóm mã có thể tái sử dụng vào một nơi.
		
		module Study
		end

		class MyClass
		  include Study
		end

		my_obj = MyClass.new
